### Анализ и тестирование платформы онлайн-торговли

#### Обзор выполненной работы

#### Выполнено:

Анализ требований — выявлены противоречия, неполнота, риски

Тестирование UI-формы — функциональные, сетевые и UX-проверки

Тестирование API — аутентификация, CRUD, негативные сценарии

Интеграционные сценарии — выявлены критические расхождения с бизнес-требованиями

Оформление баг-репортов — задокументированы найденные дефекты

Предложения по улучшению — рекомендации для продукта и процессов

Что не сделано: автоматизированные тесты (требовались опционально).
